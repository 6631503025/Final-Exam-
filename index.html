ไฟล์ `index.html` ในโปรเจคนี้ถูกแทนที่ด้วยหน้าแนะนำเพื่อความปลอดภัย: แอปจะไม่เรนเดอร์จนกว่าจะรัน dev server หรือเปิดหน้า build

  • รัน dev server (แนะนำ): เปิดเทอร์มินัลที่โฟลเดอร์โปรเจคแล้วรัน npm install ตามด้วย npm run dev แล้วเปิด http://localhost:5173
  • ดูโค้ดอย่างเร็ว: เปิดไฟล์ code-view.html ในโปรเจค (ดับเบิ้ลคลิก) เพื่อดูซอร์สไฟล์ในเบราว์เซอร์ โดยไม่ต้องรันเซิร์ฟเวอร์
  • ดูโปรดักชัน (built): รัน npm run build แล้ว serve โฟลเดอร์ dist/

หากคุณต้องการผมสามารถคืนค่า `index.html` ให้เป็นไฟล์ entry ของ React ได้หรือทำการแยกไฟล์ placeholder แยกชื่ออื่นแทน — บอกผมได้เลย
